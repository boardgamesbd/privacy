Introduction

This Privacy Policy describes how Fast Proxies for Telegram (the "App") handles information. Our primary goal is to provide a simple and secure way to access MTProto proxies while maintaining user anonymity.

1. Important: Not a VPN

The App is a utility tool for managing MTProto proxy configurations. Please be aware:

  • The App does not provide VPN (Virtual Private Network) services.
  • The App does not tunnel your entire device's internet traffic.
  • It only helps you find and add proxy settings to your messaging client.

2. Data Collection

We believe in privacy. The App is designed to work without requiring you to provide personal data:

  • No Accounts: You don't need to register or log in.
  • No Personal Info: We don't collect names, emails, or phone numbers.
  • No Logs: We do not track which proxies you use or your messaging activity.

3. Third-Party Services

To provide functionality and analyze app performance, we use the following services:

3.1. GitHub

The proxy list is fetched from a public repository on GitHub. This request is anonymous and retrieves only the necessary configuration file.

→ Read GitHub Privacy Statement

  • Analytics: Aggregated data on how users interact with the app.

To keep the app free, we show ads via Yandex. YAN may use anonymous advertising identifiers (such as GAID) to provide relevant content.
